// 001 · Featured
Fully Autonomous Heavy-Lift UAV for Disaster Response

A fully custom heavy-lift UAV built from the ground up for flood relief missions. The system autonomously scans large regions, detects survivors using onboard perception, and delivers payloads — all without manual control.

  • 1000 mm coaxial airframe — 3D-printed + carbon fiber construction
  • 22 min flight time with 5 kg payload capacity
  • Pixhawk + Raspberry Pi companion computer autonomy stack
  • Mission planning from KML files via QGroundControl
  • Servo-actuated payload delivery mechanism
  • 4G LTE long-range communication link

// 002 · Research
Communication-Aware Swarm Policy Learning

IEEE RA-L target paper co-designing MARL coordination policy with knowledge of the ESP-NOW mesh substrate. Tests the hypothesis that substrate-aware policy training outperforms naive deployment across 2–3 quadrotors in real flight.

  • Decentralized MARL training on HPC cluster (CTDE paradigm)
  • ESP-NOW mesh characterization — latency, PDR, topology mapping
  • ArduPilot + MAVSDK integration for real hardware validation
  • GPU-accelerated swarm simulation (CUDA/Thrust FDTD-style)
  • Targeting July 2026 IEEE RA-L submission

// 003 · EW / SDR
Autonomous Electronic Warfare Defense Node (AEWDN)

A closed-loop agentic AI system modeled on the OODA doctrine for detecting and responding to RF threats — jamming, GPS spoofing, and protocol hijacking — using HackRF One and a real-time signal processing pipeline.

  • Real-time FFT waterfall with 8-feature RF extraction pipeline
  • IQ capture and Python/NumPy signal classification
  • TDOA-based jammer geolocation (DSIN variant)
  • GAN-based adversarial RF attack synthesis (ARGOS variant)
  • LSTM onboard link health monitor (ORCA variant)
